Business Chat Widget Test Page

This page demonstrates the Business Chat Widget integration. You should see a chat button in the bottom right corner of the page.

How to Add to Your Website

Copy and paste the following code just before the closing </body> tag on your website:

<script src="https://widget-chat-app.netlify.app/chat.js"></script>
<script>
  new BusinessChatPlugin({
    uid: 'YOUR_USER_ID'
  });
</script>

Configuration Options

The widget accepts the following configuration options:

  • uid (required): Your user ID from the dashboard
  • position (optional): Where to position the widget. Options: 'bottom-right' (default), 'bottom-left', 'top-right', 'top-left'

Example with all options:

<script src="https://widget-chat-app.netlify.app/chat.js"></script>
<script>
  new BusinessChatPlugin({
    uid: 'YOUR_USER_ID',
    position: 'bottom-left'
  });
</script>
